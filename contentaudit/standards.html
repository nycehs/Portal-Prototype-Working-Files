Copy standards

Be brief. The purpose of titles is to convey all the necessary information efficiently - users are deciding what to click on by scanning these. Eliminate extra words.

The most important words should be first. Extra information should be in parenthesis. Example:

  • OLD: Adults Reporting Heavy Drinking
  • NEW: Heavy drinking (adults)

Use sentence case. Capitalize only the first letter, not each word. That's usually easier to read.

Age should be styled as:

  • Indicator name (age x-y)
  • Indicator name (under age X)
  • Indicator name (age Y+)

If geography is part of the indicator name, add it afterward. This is consistent with "put the most important words first." For example:

  • OLD: Asthma Emergency Department Visits (Children 0 to 4 Yrs Old) by NTA
  • NEW: Asthma ED visits (age 0 to 4), by NTA
