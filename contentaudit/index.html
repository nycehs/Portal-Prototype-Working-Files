Metadata standards

Field Standards
Indicator name The name of the indicator
Label This is used for displaying when organized under headers that show comon information common across different indicators (like asthma ed visits, hospitalizations, prevalence)
Short name This is usued on the intial display of the Neighborhood Reports. It needs to be short to fit on the page. It does not need details - just enough to distinguish the general topic from other indicators.
Description A brief description (2-3 sentences maximum!) that offers any necessary details about the indicator and why it's important or what it indicates.

Copy standards

The most important words should be first. Extra information should be in parenthesis. Example:

  • OLD: Adults Reporting Heavy Drinking
  • NEW: Heavy drinking (adults)

Use sentence case. Capitalize only the first letter, not each word. That's usually easier to read.

Age should be styled as:

  • Indicator name (age x-y)
  • Indicator name (under age X)
  • Indicator name (age Y+)
